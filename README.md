# 안정호 — Brand & Growth Marketing Leader

다크 옐로 톤의 단일 페이지 스크롤 포트폴리오. 17년 차 디지털 / 그로스 / 커머스 마케팅 리더 안정호의 마스터 포트폴리오.

태그라인: **광고를 넘어, 사업을 키웁니다.**

🔗 라이브: https://leoahn-xoxo.github.io

## 구성

- **Hero** — 키네틱 워드 슬롯(사업을 / 매출을 / 브랜드를 / 팀을 키웁니다) + 광고주 마퀴
- **About** — 운영자가 아니라 사업을 키운 사람 + 진단 우선 일하는 방식
- **핵심역량 4기둥** — 이커머스 / 그로스, 캠페인 프로모션 기획(IMC), 미디어 사업 / 수익화, 데이터 + AI
- **케이스** — 필터 가능한 13개 케이스 그리드 (모든 수치에 기간 / 모수 명시)
- **Career** — 커리어 타임라인
- **Contact**

## 로컬에서 보기

빌드 없는 정적 사이트. 아무 정적 서버로 열면 됩니다.

```bash
python -m http.server 5173
# 또는
npx serve .
```

브라우저에서 `http://localhost:5173` 접속.

## 기술

순수 HTML / CSS / JS (무빌드). 폰트는 Google Fonts(Big Shoulders Display, Space Mono, Noto Sans KR) + Pretendard(jsdelivr CDN). 한글은 Pretendard, CDN 차단 시 Noto Sans KR로 백업. GitHub Pages 배포.
