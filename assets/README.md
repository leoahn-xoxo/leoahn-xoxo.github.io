# assets — 넣어야 할 이미지

파일을 정확한 이름으로 넣으면 사이트에 자동 반영됩니다.
이미지가 없으면 프로젝트명 플레이스홀더로 자연스럽게 대체되고(깨진 이미지 안 보임), **케이스 이미지는 골드 듀오톤 처리**가 자동 적용돼 톤이 하나로 묶입니다.

## 이미 들어간 것
- `assets/profile.jpg` — 컨택 섹션 둥근 프로필(흑백 처리)
- `assets/app-yeongdaneo.png` — 데이터·AI 영단어찾기 앱 아이콘
- `assets/cases/samsung.jpg` — 삼성닷컴 통합 대시보드
