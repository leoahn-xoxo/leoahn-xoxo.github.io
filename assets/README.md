# assets — 넣어야 할 이미지

파일을 정확한 이름으로 넣으면 사이트에 자동 반영됩니다.
이미지가 없으면 프로젝트명 플레이스홀더로 자연스럽게 대체되고(깨진 이미지 안 보임), **모든 이미지는 골드 듀오톤 처리**가 자동 적용돼 톤이 하나로 묶입니다.

> 이력서 PDF 다운로드 버튼은 공개 사이트 특성상 제거됨. 연락은 컨택 섹션의 이메일/전화/LinkedIn으로.

## 1) 커버 (히어로 배경)
- `assets/cover.jpg` ← 첫 화면 풀스크린 배경. 가로형 고해상도(1600px+ 폭) 권장. 캠페인 키비주얼이나 무드 컷.
